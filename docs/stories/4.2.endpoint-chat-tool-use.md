# Story 4.2: Endpoint de chat com tool use + MCP Supabase

## Contexto
Epico 4. Cobre FR12 (multi-arquivo) e FR14a (MCP Supabase). Componente 7 da arquitetura. Reusa `providers/claude.js` com 2 params novos opcionais (`mcpConfigPath`, `cwd`).

## Descricao
Como Diogo, quero um endpoint que envia uma mensagem ao Claude com contexto do projeto, tool use de filesystem (`read_file`, `write_file`, `list_dir`) escopadas ao `cwd` do projeto e MCP do Supabase, para o agente operar no codigo real.

## Tarefas tecnicas
1. [ ] Modificar `lib/workers/providers/claude.js` adicionando params opcionais `mcpConfigPath` e `cwd` (passados como `--mcp-config` e `cwd` do spawn). Backwards-compatible.
2. [ ] Criar `lib/builder/agent.js` exportando `sendMessage({ projectId, message })`: monta system prompt usando `indexer.buildContext`, spawna `claude` com `cwd=projectPath`, `--mcp-config <projectPath>/.mcp.json`, tools de filesystem.
3. [ ] Tools de filesystem **escopadas**: `read_file`/`write_file`/`list_dir` so resolvem paths dentro de `projectPath`. Path traversal (`../`) rejeitado.
4. [ ] Confirmacao explicita para operacoes destrutivas via Supabase MCP (drop, delete em massa) — flag passada no system prompt.
5. [ ] Endpoint `POST /api/builder/projects/:id/agent/message` com body `{ message }`; resposta SSE streaming via Worker Bus existente.

## Arquivos envolvidos
- `lib/workers/providers/claude.js` - modificar (params opcionais)
- `lib/builder/agent.js` - criar
- `lib/builder/router.js` - modificar (rota agent/message)

## Definition of Done
- [ ] Mensagem "liste os arquivos do projeto" retorna a arvore via tool `list_dir`
- [ ] Mensagem "leia app/agents/page.tsx" retorna o conteudo via `read_file`
- [ ] Tentar `read_file('../../etc/passwd')` retorna erro (escope)
- [ ] Spawn inclui `--mcp-config` apontando para `.mcp.json` do projeto
- [ ] Streaming SSE funciona (tokens chegam progressivamente)
- [ ] Hub atual (chat principal, workers) continua intacto
- [ ] Sem erros no console
- [ ] F5 preserva estado (sessao do agente persiste)
