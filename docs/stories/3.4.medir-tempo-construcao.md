# Story 3.4: Medir tempo de construcao via builder vs estimativa manual

## Contexto
Epico 3. Cobre AC do epico ("tempo total documentado") e G6 do PRD ("menor que escreveria manualmente"). Story de validacao/registro, nao codigo. Gate de seguir para o Epico 4.

## Descricao
Como Diogo, quero medir e registrar o tempo gasto construindo a tela via builder e comparar com a estimativa de construir manualmente em Next.js, para decidir se o builder atinge G6 ou se precisa voltar pra prancheta antes do Epico 4.

5. [ ] Postar resumo em `docs/stories/3.4-resultado.md`: status do gate + decisao + acoes seguintes.
