# Story 2.3: Canvas com drag-and-drop, selecao, reordenacao e delete

## Contexto
Epico 2. Cobre FR2 + FR3 parcialmente (selecao). Componente 8 — coracao da UI do builder. Coordena com Story 2.1 (patch JSON) e 2.2 (palette).

## Descricao
Como no-coder, quero arrastar componentes da palette para o canvas, soltar dentro de containers, selecionar, reordenar e deletar, para montar a estrutura visual da pagina.

## Tarefas tecnicas
1. [ ] Criar `public/builder/canvas.js` que renderiza a `tree` da pagina ativa do `builder.json`. Cada no vira um `<div>` ou Web Component com `data-path="/pages/0/tree/children/0/..."`.
2. [ ] Implementar drop handler: ao receber componente da palette, calcular `path` de destino (drop entre filhos de um Container), montar JSON Patch `{ op: "add", path, value: {...} }` e enviar para `/api/builder/projects/:id/schema/patch`.
3. [ ] Selecao: clique em no destaca borda + emite evento `builder:selection` com path; armazenar em estado local `selectedPath`.
4. [ ] Reordenacao via drag interno: arrastar no ja existente para nova posicao gera patch `move`. Tecla Delete em selecao gera patch `remove`.
5. [ ] Optimistic UI: aplicar mutacao localmente antes do servidor responder; em erro do PATCH, fazer rollback e exibir toast.

## Arquivos envolvidos
- `public/builder/canvas.js` - criar (render + drag/drop/select/reorder/delete)
- `public/builder/index.html` - modificar (coluna central do canvas)
- `public/builder/style.css` - modificar (highlight de selecao, drop indicators)
- `public/builder/app.js` - modificar (estado global da pagina ativa + selecao)

## Definition of Done
- [ ] Arrastar Botao da palette dentro de Container do canvas adiciona o no e gera patch correto
- [ ] Selecao destaca visualmente (borda azul) e expoe path no estado
- [ ] Drag interno move o no para nova posicao
- [ ] Tecla Delete remove o no selecionado
- [ ] Optimistic UI: alteracao visivel em <= 100ms (NFR1 budget)
- [ ] Rollback em caso de erro do PATCH
- [ ] Sem erros no console
- [ ] F5 preserva estado (reler `builder.json`)
