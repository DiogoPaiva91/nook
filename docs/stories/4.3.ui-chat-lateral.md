# Story 4.3: UI de chat lateral no builder com historico por projeto

## Contexto
Epico 4. Cobre parte da UI de FR11/FR12. Painel lateral integrado ao shell do builder, consumindo o endpoint da Story 4.2.

## Descricao
Como Diogo, quero um painel de chat lateral no builder que mantem historico por projeto, mostra streaming de tokens e tool calls em tempo real, para conversar com o agente sem sair do canvas.

## Tarefas tecnicas
1. [ ] Criar `public/builder/chat.js`: painel colunar (toggle aberto/fechado), input multiline, lista de mensagens. Persistir historico em SQLite via tabela `builder_chat_messages (id, project_id, role, content, created_at)`.
2. [ ] Adicionar tabela em `lib/builder/db.js` (migration idempotente).
3. [ ] Endpoint `GET /api/builder/projects/:id/agent/messages` para historico; `POST /agent/message` (Story 4.2) ja envia.
4. [ ] Renderizar SSE: tokens streaming + tool calls renderizados em "cards" colapsaveis (read_file mostra path; write_file mostra path + diff abreviado — diff completo na Story 4.4).
5. [ ] Indicador "agente pensando" quando aguardando primeiro token; botao "parar" para abortar (kill subprocess).

## Arquivos envolvidos
- `public/builder/chat.js` - criar
- `public/builder/index.html` - modificar (coluna lateral chat)
- `public/builder/style.css` - modificar (estilos chat + tool cards)
- `lib/builder/db.js` - modificar (tabela messages)
- `lib/builder/router.js` - modificar (rotas messages history + abort)
